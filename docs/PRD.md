## 1. 제품 개요

### 1.1 제품명
**노무뚝딱** (Nomu Ttuktak)

### 1.2 제품 비전
소규모 사업장(1~10인)을 위한 **원스톱 노무서류 작성 및 급여 관리 서비스**

### 1.3 타겟 사용자
| 구분 | 설명 |
|------|------|
| **1차 타겟** | 직원 1~5명 소규모 사업장 대표 |
| **2차 타겟** | 스타트업, 프리랜서 고용 사업주 |
| **3차 타겟** | 노무사/세무사 사무실 (고객용) |

### 1.4 핵심 가치 제안 (Value Proposition)
1. **간편함**: 복잡한 노무 지식 없이 클릭만으로 서류 작성
2. **정확함**: 2026년 최신 법률/요율 자동 반영
3. **절감**: 비과세 항목 최적화로 4대보험료 절감
4. **연동**: 직원 정보 한 번 입력 → 모든 서류에 자동 반영

## 4. 기능 요구사항 상세

### 4.1 회원 시스템 (Phase 2)

#### FR-001: 회원가입
- **설명**: 사용자가 이메일로 회원가입할 수 있어야 한다
- **수락 기준**:
  - 이메일 형식 검증
  - 비밀번호 최소 8자, 영문+숫자 포함
  - 이메일 인증 필수
  - 약관 동의 체크박스

#### FR-002: 로그인
- **설명**: 등록된 사용자가 로그인할 수 있어야 한다
- **수락 기준**:
  - 이메일/비밀번호 로그인
  - 소셜 로그인 (Google, Kakao)
  - "로그인 유지" 옵션
  - 비밀번호 찾기 기능

#### FR-003: 회사 프로필
- **설명**: 사용자가 회사 정보를 관리할 수 있어야 한다
- **수락 기준**:
  - 회사명, 사업자등록번호, 대표자명
  - 주소, 연락처
  - 업종, 근로자 수
  - 로고 업로드 (선택)

### 4.2 임금대장 (Phase 1)

#### FR-010: 임금대장 자동 생성
- **설명**: 급여명세서 발급 시 임금대장에 자동 기록
- **수락 기준**:
  - 월별/연도별 조회
  - 직원별 급여 이력
  - 총 지급액/공제액 합계
  - Excel 다운로드

### 4.3 4대보험 신고 (Phase 1)

#### FR-020: 취득신고서
- **설명**: 신규 직원 4대보험 취득신고서 생성
- **수락 기준**:
  - 국민연금/건강보험/고용보험 각각
  - 직원 정보 자동 입력
  - 인쇄/PDF 다운로드

#### FR-021: 상실신고서
- **설명**: 퇴사 직원 4대보험 상실신고서 생성
- **수락 기준**:
  - 상실사유 선택
  - 상실일 입력
  - 인쇄/PDF 다운로드

3. **소규모 특화**: 1~10인 사업장 최적화
